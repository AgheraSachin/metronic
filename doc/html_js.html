Javascripts

initialize.js is the most important script that handles most of the base layouts initialization. The layout script are located within each demo's scripts folder; [metronic]/theme/html/[demo]/src/js/layout/initialize.js

Method Description
KTLayoutHeader('kt_header', 'kt_header_mobile')

Init Desktop & Mobile Headers
KTLayoutHeaderMenu('kt_header_menu', 'kt_header_menu_wrapper')

Init Header Menu
KTLayoutHeaderTopbar('kt_header_mobile_topbar_toggle')

Init Header Topbar For Mobile Mode
KTLayoutBrand('kt_brand')

Init Brand Panel For Logo
KTLayoutAside('kt_aside')

Init Aside
KTLayoutAsideToggle('kt_aside_toggle')

Init Aside Menu Toggle
KTLayoutAsideMenu('kt_aside_menu')

Init Aside Menu
KTLayoutSubheader('kt_subheader')

Init Subheader
KTLayoutContent('kt_content')

Init Content
KTLayoutFooter('kt_footer')

Init Footer
KTLayoutScrolltop('kt_scrolltop')

//////////////////////////////////////////// // Init Scrolltop
KTLayoutStickyCard('kt_page_sticky_card')

Init Sticky Card
KTLayoutStretchedCard('kt_page_stretched_card')

Init Stretched Card
KTLayoutExamples()

Init Code Highlighter & Preview Blocks(used to demonstrate the theme features)
KTLayoutDemoPanel('kt_demo_panel')

Init Demo Selection Panel
KTLayoutChat()

Init Chat App(quick modal chat)
KTLayoutQuickActions('kt_quick_actions')

Init Quick Actions Offcanvas Panel
KTLayoutQuickNotifications('kt_quick_notifications')

Init Quick Notifications Offcanvas Panel
KTLayoutQuickPanel('kt_quick_panel')

Init Quick Offcanvas Panel
KTLayoutQuickUser('kt_quick_user')

Init Quick User Panel
KTLayoutQuickSearch('kt_quick_search')

Init Quick Search Panel

Init Quick Cart Panel

Init Search For Quick Search Dropdown
