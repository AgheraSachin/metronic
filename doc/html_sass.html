SASS

Layout

Each demo has own base style(the layout and components) [metronic]/theme/html/[demo]/src/sass/style.scss that will be compiled into the demo's base css bundle style.bundle.css.

Mixins

When you customize or code in Metronic's SASS environment you can use Metronic's globally available base mixins. All available mixins can be found here: [metronic]/theme/html/demo1/src/sass/base/mixins/*

SASS Functions

You can use Metronic's globally available SASS functions in your custom codes. All available functions can be found here: [metronic]/theme/html/demo1/src/sass/base/functions/*

Metronic uses nested map list to structure the config options with special custom function get() to access the nested config structure. You can access to an option by using get() function as shown below: 

transition: get($aside-config, minimize, transition);
